Krishi Sahay: Multilingual AI Voice Assistant for Farmers

Overview Link to heading

Krishi Sahay is a telephony-based AI assistant designed to bridge the digital divide for farmers. By using interactive voice response (IVR), the system allows users to access complex agricultural data and book logistics services without needing a smartphone or internet connectivity, essentially acting as an offline AI interface for the end-user.

My Role Link to heading

  • Backend development and system architecture using Flask.
  • Integration of Gemini AI for natural language processing and entity extraction.
  • Telephony workflow design using Twilio to handle voice-to-text and text-to-voice loops.
  • Implementation of a multilingual support system using Azure Speech services.

Impact Link to heading

  • Created a hands-free booking system for agricultural logistics and equipment.
  • Enabled offline AI accessibility, allowing users with basic feature phones to interact with advanced LLMs via standard voice calls.
  • Developed a scalable complaint and registration system using Firestore for efficient data management.
